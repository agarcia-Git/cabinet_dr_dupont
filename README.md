# Cabinet Dr. Dupont — Application de gestion des rendez-vous

## Description
Application web de gestion des rendez-vous pour le Cabinet Dr. Dupont et son équipe.

### Front Office (patients)
- Page d'accueil avec présentation du cabinet et horaires
- Prise de rendez-vous en ligne
- Page des services proposés
- Page actualités avec détail des articles
- Page à propos (Dr. Dupont et son équipe)

### Back Office (administration)
- Authentification sécurisée
- Tableau de bord
- Gestion des rendez-vous (confirmer, annuler)
- Gestion des patients 
- Gestion des services 
- Gestion des actualités 
- Gestion des horaires

## Technologies utilisées
- **Backend** : PHP 8 (POO, architecture MVC)
- **Base de données** : MySQL / PDO
- **Frontend** : HTML5, CSS3, Bootstrap 5, JavaScript
- **Serveur local** : WAMP
- **Versioning** : Git / GitHub
